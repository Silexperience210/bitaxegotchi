BitaxeGotchi — Flasher complet via esptool (esp-web-flasher)

Cette page vous permet d'utiliser le flasher complet construit autour d'esptool dans le navigateur. Deux modes :

  • Utiliser la version publique hébergée par Espressif (rapide, pas d'installation).
  • Héberger esp-web-flasher localement dans ce dépôt pour un flasher intégré (méthode recommandée pour production / personnalisation).
Ouvrir dans un nouvel onglet Instructions d'intégration locale

Si vous voulez un flasher "tout-en-un" directement dans ce dépôt, suivez les instructions du bouton "Instructions d'intégration locale" pour ajouter esp-web-flasher (submodule ou copie des fichiers dist). Une fois les fichiers dist présents localement, vous pouvez remplacer l'iframe ci‑dessous par la version locale en modifiant le src.

Note importante :
  • Le flasher utilise l'API Web Serial : fonctionne uniquement dans les navigateurs qui la supportent (Chrome, Edge). Safari n'a pas encore le support complet.
  • Pour flasher un ESP32/ESP8266, mettez votre périphérique en bootloader (si besoin) et suivez les instructions de l'UI esp-web-flasher.
  • Si vous voulez que je fasse l'intégration complète (ajout des fichiers dist et PR), dites-le et je l'ajouterai directement au dépôt.

Si l'iframe ne fonctionne pas

Ouvrez le flasher distant dans un nouvel onglet en cliquant sur "Ouvrir dans un nouvel onglet". Si vous préférez héberger localement :

1) Ajouter esp-web-flasher en sous-module :
   git submodule add https://github.com/espressif/esp-web-flasher.git webflasher/esp-web-flasher-upstream
   cd webflasher/esp-web-flasher-upstream
   npm install
   npm run build
   # copier le contenu de dist/ dans webflasher/esp-web-flasher-dist (ou servir directement)
2) Remplacer l'iframe src par la version locale (ex : ./esp-web-flasher-dist/index.html)

Souhaitez-vous que j'ajoute l'intégration complète au dépôt (copie des fichiers dist + page locale) et que j'ouvre une PR ?
