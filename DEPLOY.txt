ÉTAPES DE DÉPLOIEMENT:
======================

1. Crée un nouveau repo GitHub

2. Upload TOUS ces fichiers à la racine:
   - index.html
   - manifest.json
   - bootloader.bin (tes binaires)
   - partitions.bin (tes binaires)
   - boot_app0.bin (tes binaires)
   - firmware.bin (tes binaires)

3. Active GitHub Pages:
   - Settings → Pages
   - Source: "Deploy from a branch"
   - Branch: "main" + "/ (root)"
   - Save

4. Attends 1-2 minutes

5. Ta page sera accessible à:
   https://TON-USERNAME.github.io/TON-REPO/


IMPORTANT:
==========
- Les fichiers .bin doivent être à la racine (même niveau que index.html)
- Les noms des fichiers .bin doivent correspondre exactement au manifest.json
- Si tes fichiers .bin ont d'autres noms, modifie le manifest.json
